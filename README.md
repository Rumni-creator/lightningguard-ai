## 🔥 What This Project Does

LightningGuard-AI is a **Bitcoin & Lightning Network wallet analyzer** powered by **AI-style risk scoring**.  
It allows users to input a wallet address and get a simple, actionable risk assessment:

- ✅ **Low Risk** – typical activity, no red flags  
- ⚠️ **Suspicious** – unusual patterns, flagged for review  
- 🚨 **High Risk** – very active or unusual wallets  

The app uses real **Bitcoin API data (mempool.space)** to fetch wallet stats and applies **rule-based AI scoring** to flag potential risks.

---

## 🧠 How It Works (Simple Version)

1. User inputs a **Bitcoin address**  
2. The backend fetches data from **mempool.space API**  
3. Analyzes patterns such as:
   - Number of transactions  
   - Funded vs spent amounts  
   - Wallet activity  
4. Returns a **risk assessment**  
5. (Optional) Could integrate Lightning Network insights for micro-payments
